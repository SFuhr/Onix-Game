# DearTown!
## by Stanislav Khrustalenko
### Специально для Onix-Systems

Постарался воссоздать максимально точно игровой процесс из видео. Размытую камеру сзади добавил тоже чтобы соответствовало видео, но больше ради прикола. Весь код написан мною, иногда с помощью гугла.

Хотя обычно я так не делаю, но в игре есть проблема, движение влево-вправо зависит фреймрейта. Я честно пытался, но не осилил это пофиксить по-человечески. Проблема заложена в алгоритме движения по пикселям, и слишком поздно я об этом подумал. Но в остальном багов быть не должно. Игра с 60 фпс работает идеально. В конце концов, даже bethesda позволяет себе делать фреймозависимые игры, чем я хуже!? Хех.

Надеюсь, вам понравится!
