# DearTown!
## by Stanislav Khrustalenko
### Специально для Onix-Systems

Постарался воссоздать максимально точно игровой процесс из видео. Размытую камеру сзади добавил тоже чтобы соответствовало видео, но больше ради прикола. Весь код написан мною, иногда с помощью гугла.

Есть проблема с зависимым от фреймрейта движением влево-вправо. Очень поздно я об этом подумал, да я не уверен, как в такой реализации это пофиксить. Только если с нуля по-другому переделывать. Игра работает как задумано в 60 фпс.

Надеюсь, вам понравится!
